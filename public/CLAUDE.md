# public 目录说明

本目录存放项目的公共资源，按用途拆分为文档、静态文件、测试相关目录。

## 目录结构

```text
public/
├── docs/              # 项目文档
├── storage/           # 静态文件目录
│   ├── document/      # 文档类文件
│   └── image/         # 图片类文件
└── test/              # 单元测试与用例测试目录
```

## 目录职责

- **docs/** - 项目文档目录，存放设计方案、说明文档、技术资料
- **storage/** - 静态文件目录
  - **document/** - 文档类静态文件
  - **image/** - 图片类静态文件
- **test/** - 单元测试、用例测试相关目录

## 使用约定

- 涉及项目说明、设计方案、技术资料时，优先查看 `docs/`
- 涉及上传、访问、归档静态资源时，按文件类型放入 `storage/document/` 或 `storage/image/`
- 涉及测试资源、测试用例、测试数据时，放入 `test/`
- `public/` 目录只存放公共资源，不放业务实现代码
